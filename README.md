# Python String Swap Without Temporary Variable

## Description

This project contains a Python program that allows the user to enter two strings and exchange their contents **without using an auxiliary variable**.

The program demonstrates a basic algorithmic technique using:
- string concatenation
- string slicing
- custom length calculation

## Objective

Write a Python program that:
1. Reads two strings from the user.
2. Swaps their values without creating a third variable.
3. Displays the values before and after the swap.
